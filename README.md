# TreeGrow - Tree-Plant Reward Program 🌱

## Overview

TreeGrow is an innovative tree-planting reward program built on the Stacks blockchain that incentivizes environmental restoration through smart contracts. The system rewards participants for planting and maintaining trees, creating a sustainable economy around reforestation and carbon sequestration.

## Problem Statement

Climate change and deforestation are critical environmental challenges that require immediate action. Current tree-planting initiatives lack:

- **Accountability**: No reliable tracking of planting activities
- **Incentivization**: Limited rewards for environmental stewardship  
- **Transparency**: Difficulty verifying actual tree planting and survival
- **Long-term Commitment**: Poor mechanisms for ongoing tree care and monitoring

## Solution

TreeGrow addresses these challenges by creating a blockchain-based incentive system that:

1. **Rewards Planting**: Participants earn GRW (GrowReward) tokens for verified tree planting
2. **Tracks Growth**: Immutable records of tree planting locations and progress
3. **Enables Monitoring**: Regular check-ins to verify tree survival and health
4. **Creates Community**: Decentralized network of planters, validators, and environmental supporters

## Core Features

### 🌳 Tree Planting System
- Submit tree planting activities with location and proof
- Earn GRW tokens based on tree species and survival probability
- Track individual and community planting statistics
- Generate planting certificates and achievements

### 🏛️ Reward Distribution
- Dynamic token rewards based on tree type and environmental impact
- Bonus rewards for native species and biodiversity enhancement
- Long-term incentives for tree survival milestones
- Community validation of planting activities

### 📊 Environmental Impact Tracking
- Monitor total trees planted by the community
- Calculate estimated carbon sequestration
- Track biodiversity improvements and habitat creation
- Generate environmental impact reports

### 🛡️ Verification System
- Community-based validation of tree planting claims
- Photo/GPS verification requirements
- Regular survival check-ins and health assessments
- Integration with environmental monitoring systems

## Smart Contracts

### 1. Tree Registry Contract (`tree-registry.clar`)
Manages the core tree planting and tracking system:
- Tree planting submission and verification
- Token minting for verified plantings
- Location tracking and mapping
- Species categorization and rewards
- Survival milestone tracking

### 2. Reward Distribution Contract (`reward-distribution.clar`)
Handles token distribution and incentive mechanisms:
- Reward calculation based on tree species and impact
- Milestone-based bonus distribution
- Community validator rewards
- Staking mechanisms for validators
- Achievement and badge system

## Technology Stack

- **Blockchain**: Stacks (Bitcoin Layer 2)
- **Smart Contracts**: Clarity language
- **Development**: Clarinet framework
- **Testing**: Vitest framework
- **Version Control**: Git/GitHub

## Getting Started

### Prerequisites
- Clarinet CLI installed
- Node.js and npm
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/misbauchoji-art/TreeGrow.git
   cd TreeGrow
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Check contracts:
   ```bash
   clarinet check
   ```

4. Run tests:
   ```bash
   npm test
   ```

## Usage

### For Tree Planters

1. **Register**: Create a profile in the TreeGrow system
2. **Plant**: Plant trees and submit verification details
3. **Document**: Provide location, species, and photo evidence
4. **Earn**: Receive GRW tokens for verified plantings
5. **Monitor**: Track tree survival and earn milestone rewards

### For Validators

1. **Stake**: Stake GRW tokens to become a validator
2. **Verify**: Review and validate tree planting submissions
3. **Monitor**: Conduct periodic tree health check-ins
4. **Earn**: Receive validation rewards for accurate assessments

### For the Community

1. **Support**: Purchase and hold GRW tokens to support the ecosystem
2. **Sponsor**: Fund specific planting projects or locations
3. **Track**: Monitor global impact and reforestation progress
4. **Participate**: Join community governance and decision-making
